# Extension Icons

Place your extension icons here:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

For now, you can use placeholder icons or generate them using an icon generator.

Temporary SVG icon you can convert to PNG:

```svg
<svg width="128" height="128" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" fill="#0066cc" rx="20"/>
  <text x="64" y="80" font-size="64" text-anchor="middle" fill="white">✍️</text>
</svg>
```
